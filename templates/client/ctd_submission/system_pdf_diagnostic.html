Guide d'Installation

Pour une extraction PDF optimale, installez les bibliothèques suivantes :

Recommandé (extraction complète)
pip install PyMuPDF
PyMuPDF offre la meilleure extraction avec tableaux, images et structure préservée.
Excellent pour tableaux
pip install pdfplumber
pdfplumber excelle dans l'extraction de tableaux structurés.
Alternative robuste
pip install pdfminer.six
pdfminer est une alternative stable et fiable.
Fallback basique
pip install PyPDF2
PyPDF2 pour extraction de base en dernier recours.
