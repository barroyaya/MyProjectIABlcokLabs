Structure du JSON Global ×

Le JSON global contient toutes les annotations consolidées pour ce document, organisées de manière structurée :

document_info: Métadonnées du document

consolidated_annotations: Annotations consolidées

├── by_type: Organisé par type d'annotation

└── by_page: Organisé par numéro de page

summary: Résumé généré automatiquement

statistics: Statistiques globales

Exemple de structure :

{
  "document_info": {
    "id": "12345",
    "title": "Document Example",
    "type": "Clinical Study Report",
    "pages": 42,
    "created_at": "2023-06-15T09:30:00Z"
  },
  "consolidated_annotations": {
    "by_type": {
      "adverse_event": [...],
      "patient_demographics": [...],
      "efficacy_parameter": [...]
    },
    "by_page": {
      "1": [...],
      "2": [...]
    }
  },
  "summary": "This document contains...",
  "statistics": {
    "total_annotations": 127,
    "annotated_pages": 28,
    "coverage": "67%"
  }
}
